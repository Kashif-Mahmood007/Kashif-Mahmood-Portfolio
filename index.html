PeanutInfo – A Deep Learning Solution

Project Overview:
To tackle the increasing challenges faced in peanut agriculture, particularly in areas such as variety classification, disease diagnosis, and pest identification, I developed a comprehensive and intelligent image classification system dedicated to peanut crop analysis. This innovative solution is designed to support farmers, agricultural researchers, and agritech platforms by providing accurate and reliable identification of peanut types, harmful pests, and plant diseases. By integrating state-of-the-art deep learning techniques and advanced image processing methods, the system aims to revolutionize traditional agricultural practices, enhance decision-making, and ultimately contribute to healthier crop management and improved yields.

Functional Modules:
The project is strategically structured into three core modules, each addressing a specific aspect of peanut crop analysis. The Peanut Classification module focuses on identifying and categorizing different types of peanuts based on distinct visual characteristics, assisting in quality control and variety management. The Peanut Pest Identification module is designed to detect and classify various pests that pose threats to peanut crops, enabling early intervention and minimizing potential damage. Meanwhile, the Peanut Disease Detection module specializes in diagnosing common diseases in peanut plants through detailed analysis of leaf and crop imagery, supporting timely and accurate treatment decisions.
To integrate these functionalities into a cohesive system, we utilized two publicly available datasets for the Peanut Pest Identification and Peanut Disease Detection modules. For the Peanut Classification module, we developed a custom dataset from scratch, capturing and curating images of various peanut types.

Advanced Deep Learning Algorithms:
The core of this project is built on advanced deep learning techniques and meticulous data engineering. To achieve high accuracy across all modules, I initially employed several state-of-the-art pretrained models, including ResNet-50 and EfficientNet-B0 and B4. However, these models did not consistently reach the desired accuracy threshold of beyond 98%. To overcome this, I leveraged more advanced architectures that combine convolutional neural networks with the cutting-edge transformer concept—specifically, the ConvNeXt variants (Tiny, Small, and Base). These models not only delivered state-of-the-art performance but also set a new benchmark in accuracy and robustness for peanut crop classification, pest detection, and disease identification.

User Interface and System Deployment:
A fully integrated web interface was developed to provide a seamless and engaging user experience. The frontend, built with HTML, CSS, Bootstrap, and JavaScript, ensures a responsive and visually intuitive design that performs smoothly across all devices. On the backend, PHP and MySQL power robust user management and efficient data handling, maintaining reliability and security. To bridge the deep learning models with the platform, a custom Flask API was implemented, enabling dynamic, real-time predictions from user-uploaded images. This architecture not only guarantees fast and accurate responses but also supports scalability and future enhancements with ease.

Academic Impact and Recognition:
A standout success of this project is the publication of three impactful research papers in prestigious, peer-reviewed international platforms: IEEE, PLOS ONE, and Sensors (MDPI). These publications not only validate the innovative methodologies and technical rigor underpinning the project but also highlight its meaningful contribution to advancing agricultural image classification technologies. This academic recognition has been a tremendous confidence booster, elevating my professional profile and inspiring me to continue driving innovation and excellence in the field.

Dimensionality Reduction Using PCA

Project Overview:
This project focused on applying Principal Component Analysis (PCA) for dimensionality reduction on the MNIST dataset, which contains over 40,000 grayscale images of handwritten digits, each represented by a 28x28 pixel grid flattened into 784 features. An initial machine learning model was trained using all 784 dimensions to establish a performance benchmark. The primary goal was to reduce the high-dimensional input space to a more computationally efficient form while retaining the most significant features necessary for accurate classification. This approach aimed to optimize both model performance and training efficiency by eliminating redundant and less informative features.

Feature Reduction and Visualization:
To identify the optimal number of features for dimensionality reduction, the commonly used “90% variance retention” rule was applied, ensuring that the reduced dataset would still capture the vast majority of meaningful information. Through analysis of the explained variance ratio, it was determined that approximately 250 principal components were sufficient to preserve over 90% of the original data variance. The PCA algorithm was then used to project the original 784-dimensional data into this reduced 250-dimensional space. A K-Nearest Neighbors (KNN) classifier was subsequently trained on the transformed dataset, achieving an impressive accuracy of over 96%, nearly equivalent to the model trained on the full feature set. To better understand the structure of the data and its separability after dimensionality reduction, I visualized the transformed data using Plotly by projecting it into both 2D and 3D spaces. These interactive visualizations provided valuable insights into class distributions and clustering behavior post-PCA.

Challenges and Insights:
One of the key challenges in this project was finding the right balance between dimensionality reduction and classification accuracy, ensuring that essential data characteristics were preserved while eliminating redundancy. The implementation, carried out using Python and scikit-learn, involved calculating and visualizing the explained variance to guide the selection of optimal principal components. After applying PCA, a KNN classifier was trained on the reduced dataset to assess performance. The results confirmed that significant dimensionality reduction could be achieved without degrading model accuracy, leading to faster training times and improved computational efficiency. This project enhanced my practical expertise in data preprocessing, dimensionality reduction techniques, interactive data visualization using Plotly, model evaluation, and developing scalable machine learning workflows.

PipeSense - A Machine Learning Solution

Project Overview:
I recently completed an end-to-end machine learning regression project aimed at predicting the condition of gas and water pipes based on various physical and environmental features. The primary objective was to build a predictive model that estimates pipe condition using factors such as Pipe Size, Diameter, Thickness, Material, Strength (MPa), Maximum Pressure (Bar), Material Loss Percentage, Thickness Loss (mm), and Temperature. This model can help utility and maintenance teams prioritize pipeline inspections and replacements, ultimately improving infrastructure management and reducing failure rates.
By leveraging regression algorithms and advanced feature engineering, the project demonstrates the practical application of machine learning in predictive maintenance and infrastructure resilience.

Key Challenges:
A major challenge in this project was achieving high prediction accuracy using a regression model. Initially, using Logistic Regression yielded unsatisfactory results, as it is not designed for continuous target prediction. To address this, I implemented a series of advanced feature engineering techniques to better prepare the dataset for modeling, including:

  • Applying the Yeo-Johnson transformation using PowerTransformer to normalize skewed feature distributions.
  • Standardizing features using StandardScaler to ensure uniform scale across variables.
  • Handling categorical variables carefully:
    • Using Ordinal Encoding for ordinal categorical features.
    • Applying One-Hot Encoding for nominal (non-ordered) categorical variables.
    • Applying Label Encoding for target variable
  • Building efficient preprocessing pipelines using ColumnTransformer and Pipeline from scikit-learn to automate and standardize all transformation steps.
  • Designing an Excel-compatible workflow to facilitate scalability and future integration.


Approach & Implementation:
To implement the solution, I followed a structured machine learning workflow. I began with thorough Exploratory Data Analysis (EDA) to understand feature distributions, correlations, and potential data quality issues. I then constructed a robust feature engineering pipeline incorporating all necessary transformations and encodings.
Using scikit-learn’s pipelines and transformers, I ensured a smooth, reusable, and efficient data preprocessing architecture. After preparing the dataset, I trained a regression model that achieved 97.4% accuracy, validated using cross_val_score to confirm the model’s generalization performance.
Once satisfied with the model’s robustness, I saved it using pickle and deployed it in a production environment, enabling real-time predictions for pipe conditions.
This project significantly enhanced my practical skills in regression modeling, pipeline development, and production deployment. It showcased the power of machine learning in predictive maintenance and provided a valuable solution for improving infrastructure resilience and operational decision-making.

The correlation heatmap and seaborn clustermap, in particular, provided a high-level view of how features were interrelated, highlighting patterns of multicollinearity. Implementation and Key Insights:
The analysis was carried out using Python, leveraging powerful libraries such as pandas for data manipulation, seaborn and matplotlib for visualization, and NumPy for numerical operations. The process involved thorough data cleaning, including handling missing values, treating outliers, and addressing skewed distributions to ensure the integrity of the analysis. One of the key challenges was maintaining both visual clarity and statistical accuracy while presenting complex relationships between features. Multivariate visualizations, especially heatmaps and pairplots, provided a comprehensive understanding of how different variables interacted to influence survival outcomes. This project significantly enhanced my proficiency in data wrangling, statistical analysis, and visual storytelling, establishing a solid analytical foundation for building robust machine learning models in future projects.

Library Management System

Project Overview:
As part of my semester project, I designed and developed a robust Library Management System (LMS) using C++. The system is tailored to meet the practical requirements of educational institutions, offering seamless book management, user authentication, and interactive features. With a strong emphasis on usability and efficiency, the LMS caters to students, teachers, and administrative staff alike. From issuing books to tracking overdue returns and calculating fines, this solution automates and simplifies the day-to-day operations of a traditional library environment.

Core System Architecture:
The system is structured around four primary classes: BookInfo, IssueBook, UserInfo, and Security. The BookInfo class manages book entries, updates, deletions, and displays, while tracking the number of available copies. The IssueBook class governs book issuing protocols—allowing a 7-day borrowing period for students and 21 days for teachers—and automatically calculates fines for late returns based on the number of overdue days. The UserInfo class maintains user records, and the Security class ensures access control through proper authentication mechanisms using username, password, and email.

User Experience & Search Capabilities:
The LMS is designed with the end-user in mind, ensuring a smooth and intuitive experience for all types of users. It facilitates comprehensive book searches using multiple criteria such as ISBN number, book title, author name, publisher, and publication date. Whether the user is a librarian managing inventory or a student looking for specific materials, the system ensures fast, accurate, and user-friendly interactions.

Security, Smoothness & Speech Integration:
To protect user data and prevent unauthorized access, a robust security module has been implemented. It includes user verification through credentials and integrates barricade logic to ensure smooth transitions and controlled flow between system modules. One of the standout features of the LMS is the integration of a speech acknowledgment system. This voice output functionality allows the system to "speak" to users—offering auditory confirmations, notifications, and instructions. This not only enhances accessibility but also creates a more interactive and engaging user experience.

Achievements & Learning Outcomes:
Developing this LMS gave me hands-on experience in applying object-oriented programming principles to solve real-world problems. It strengthened my skills in C++ programming, system design, file handling, and user interface logic. Implementing advanced features like fine calculation, voice acknowledgment, and secure user authentication taught me the importance of blending functionality with user convenience. This project stands as a complete, deployable solution and a milestone in my academic journey, demonstrating both technical depth and practical relevance.
